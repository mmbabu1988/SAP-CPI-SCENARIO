Description: 
In an e-commerce business, there is a need to send order confirmation emails to customers after they place an order. The integration scenario involves integrating the e-commerce platform with the company's email system to automatically generate and send order confirmation emails to customers. This email should also include Order Tracking Number. Please note that email should be sent only when the user opted for email notifications.

Rationale: 
By implementing this integration scenario, the e-commerce business can automate the process of sending order confirmation emails to customers, ensuring a smooth Order to Cash process and providing a better customer experience.
